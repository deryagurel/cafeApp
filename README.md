# ☕ Cafe Automation App (WIP)

Bu proje, bir kafenin günlük operasyonlarını dijitalleştirmek ve müşteri sadakatini artırmak amacıyla C# dili kullanılarak Visual Studio ortamında geliştirilmeye başlanmış bir otomasyon sistemidir. 

**Durum:** 🚧 Geliştirme Aşamasında (Work in Progress)
Projenin temel mimarisi ve algoritma altyapısı kurulmuş olup, geliştirme süreci aktif olarak devam etmektedir.

## ✨ Öne Çıkan Özellikler & Modüller
* **Müşteri Sadakat Sistemi Altyapısı:** Müşteri devamlılığını artırmak amacıyla kurgulanan **"3 kahve alana 1 krep bedava"** kampanyasının algoritmik mantığı ve kod altyapısı geliştirildi.
* **Yönetici Paneli:** Sistemin yönetimsel ayarlarının ve temel kontrollerinin yapılabileceği yönetici modülü tasarlandı.
